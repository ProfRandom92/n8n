## Monitoring RSS Feeds

### Daily Checks
1. **Feed Health:** Verify all 5 feeds are working
2. **Job Volume:** Check if expected number of jobs found
3. **Quality:** Review if filter is catching right jobs
4. **Duplicates:** Ensure no duplicate jobs in sheet

### Weekly Optimization
1. **Add new keywords** based on job market trends
2. **Remove ineffective feeds** if they don't produce results
3. **Adjust location filters** based on job availability
4. **Test new job sources** for better coverage

### Troubleshooting RSS Issues
- **Feed not updating:** Check if Indeed changed URL format
- **No jobs found:** Verify search terms are still relevant
- **XML errors:** Test feed URL directly in browser
- **Timeout errors:** Increase timeout in Make.com settings
